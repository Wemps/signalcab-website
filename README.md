# Signal Cab

**The throttle, reimagined.** Precision DCC control meets modern design — manage your fleet, command your locomotives, and run your railroad from your iPhone.

## What is Signal Cab?

Signal Cab is a DCC controller app for iOS that replaces clunky handheld throttles with an intuitive, beautifully designed interface. It connects to your existing command station over WiFi and gives you full control from your phone.

### Features

- **Multi-Throttle** — Run multiple locomotives with a swipeable carousel
- **Locomotive Library** — Full roster management with search, favorites, and import
- **Function Mapping** — Guided setup with built-in decoder defaults for major brands
- **Consist Control** — Build virtual and advanced consists with per-member config
- **128 Speed Steps** — Smooth, precise throttle with haptic feedback
- **CV Programming** — Read and write CVs directly from the app
- **Configurable Safety** — E-Stop behavior, address conflict detection, and more
